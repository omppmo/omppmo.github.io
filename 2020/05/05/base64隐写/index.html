base64隐写

因为太菜了所以写不出脚本，找到了一位师傅写的好好分析一下

原理

base64隐写只能用于有两个等号和有一个等号的情况

有两个等号的可以隐藏四位，当Base64解码时，这隐藏的四位会抹去
同理，有一个等号的能隐藏两位

正则表达式

正则表达式是一个特殊的字符序列，它能帮助你方便的检查一个字符串是否与某种模式匹配。
re 模块使 Python 语言拥有全部的正则表达式功能。

一般在正则表达式之前加r，防止字符转义
